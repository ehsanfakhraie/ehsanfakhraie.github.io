Gozar Platform

Gozar Platform is a comprehensive platform for automating the repetitive processes of festivals and community-oriented events.

General modules of this platform:

  • The module for receiving submissions (multimedia, video, audio, image, etc.)

  • Submissions processing module

  • Designing the process for submissions and managing the secretariat module

  • User profile system, team building, inviting members, and so on.

  • The specialized and public judging modules (voting)

  • The module for event registration and design.

  • VOD module

  • Sales and payment module

  • Custom Content Management module

Submission module

This module is used for receiving the works submitted by the participants in the event or festival.
Submissions can be in various formats such as different types of documents, video and multimedia submissions, audio submissions, and more. The received submissions can have various parameters and data, and this information can be completed in different forms using the forms for receiving works. Features of the submission module:

  • Form for receiving work with custom parameters

  • Multi-step forms

  • Possibility to receive multiple file types and documents in one step or separate them into different stages

  • Possibility to define different stages and descriptions for each stage in the form for receiving submissions

  • Possibility to receive video works in most of the video formats and convert them to the desired format

  • Displaying submissions in the submitter profile

  • Possibility to enable the feature of editing submissions

  • Allowing users to upload large files with the ability to pause and resume the upload in case of any issues. (Resumable Upload)

  • The ability to generate thumbnails

  • The ability to receive various formats and convert them to the desired format

Comprehensive judgement module

This module includes the following sections:

  • Public judgement

  • Specialized judgment

Features of the public judgement panel:

Public judgement is a process in which the top films of the festival are shown and users can watch the films, authenticate themselves, and vote for them.

  • Authentication with phone number

  • Displaying films randomly and in a randomized order

  • Custom player with the following features:

    • Preventing downloading submissions

    • Showing timeline

    • Displaying ads if required

  • Types of output that are suitable for the needs of the client

  • SMS panel with the ability to send corresponding text messages in the management section.

  • Various analytical outputs of the website statistics
